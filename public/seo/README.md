# SEO Verification Files

Place search engine verification files here.

## Google Search Console

1. Go to https://search.google.com/search-console
2. Add property → URL prefix → `https://ohmygold.pages.dev/`
3. Choose "HTML file" verification method
4. Download the `googleXXXXXXX.html` file
5. Place it in this directory
6. Deploy the site
7. Click "Verify" in Google Search Console
8. After verified, submit sitemap: `https://ohmygold.pages.dev/sitemap.xml`

## Bing Webmaster Tools

1. Go to https://www.bing.com/webmasters/
2. Add site → `https://ohmygold.pages.dev/`
3. Choose "HTML file" or "BingSiteAuth.xml" verification
4. Save the verification file here
5. Deploy and verify
6. Submit sitemap: `https://ohmygold.pages.dev/sitemap.xml`

## Yandex Webmaster

1. Go to https://webmaster.yandex.com/
2. Add site → `https://ohmygold.pages.dev/`
3. Choose HTML file verification
4. Save the HTML file here
5. Deploy and verify
6. Submit sitemap: `https://ohmygold.pages.dev/sitemap.xml`

## Notes

- These files must be accessible at:
  `https://ohmygold.pages.dev/seo/<filename>`
- All `.html` and `.xml` files in this directory are publicly accessible
- After verification, you can keep or remove these files
